🌡️ Temperature

Scales the raw logit scores before softmax. Higher temperature flattens the distribution (more random), lower temperature sharpens it (more deterministic).

🔢 Top-K

Keeps only the top K most probable tokens. All others are filtered out and get zero probability. Set to 0 to disable.

📊 Top-P (Nucleus Sampling)

Keeps the smallest set of tokens whose cumulative probability exceeds P. Dynamically adjusts the number of candidates based on the distribution shape.

📖 How It Works

1. Raw logits are scaled by Temperature and converted to probabilities via Softmax.

2. Top-K filters: only the top K tokens remain.

3. Top-P filters: tokens are accumulated until their sum exceeds P.

4. Remaining probabilities are renormalized to sum to 1.0.

5. A token is sampled from this final distribution.
